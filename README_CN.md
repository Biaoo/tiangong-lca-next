# 天工LCA数据平台

[English](./README.md) | [中文](./README_CN.md)

🌐 **即刻访问在线系统**  
立即开启您的LCA之旅：[https://lca.tiangong.earth](https://lca.tiangong.earth)

支持Docker本地化部署，详见[私有部署指南](https://docs.tiangong.earth/docs/deploy/local-deploy)
